# Documentation - Habit Tracker
Do Habits and play.


## APP Features

- ### Habit Features
  - Create a Habit
  - Delete a Habit 
  - Edit a Habit
  - Pause a Habit
  - Completing a Habit gives a reward, the longer the streak, the higher the reward.
